## Notes

- **39 tokens** au total (un par questionnaire)
- Les tokens sont gérables via l'interface admin Django (`/admin/` > **Accès Key Users**)
- Un key user ne voit **que** son questionnaire, pas le reste de l'application
- Les key users avec plusieurs systèmes (BENYELLES, SAID Sihem, YOUCEF ACHIRA, SAMEUR) ont un **sélecteur** pour basculer entre questionnaires
- Les tokens peuvent être désactivés individuellement depuis l'admin
- Base URL prod : `https://cartographie-si-airalgerie.onrender.com`
